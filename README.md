# 🔥 Diabolic_v17 v5.1 - OSINT Tool para Baleares

[![Version](https://img.shields.io/badge/version-5.3-red)](https://github.com/Condor2026/Diabolic_v17)
[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)
[![Python](https://img.shields.io/badge/python-3.8+-green)](https://python.org)

Herramienta OSINT que monitoriza **18 periódicos de Baleares** (Mallorca, Menorca, Ibiza, Formentera) para detectar, clasificar y visualizar patrones delictivos (robos, estafas, narcotráfico, violencia...).

## 🚀 Características

- ✅ Scraping inteligente con detección automática de URLs rotas.
- ✅ Rotación de User-Agent para evitar bloqueos.
- ✅ Clasificación automática de delitos (12+ tipos).
- ✅ Detección de conexiones entre incidentes (opción 3).
- ✅ Visualización web interactiva con gráficos dinámicos.
- ✅ Exportación a JSON y CSV.
- ✅ Menú terminal con 10 comandos.
- ✅ 100% legal y ético: solo datos públicos, sin datos personales.
